# v0.2.0 — Agent Trace Panel 与结构校验增强

## 版本目标

- 新增 Agent 执行轨迹面板，前端可视化 Agent 步骤
- 增加 `validate_tasklist_structure` 确定性质量门工具
- 支持 Agent 步骤的流式协议事件（agent-step-start / agent-step-end）
- 前端 AgentTracePanel 组件，折叠展示资源/工具细节

## 非目标

- 不做通用 Agent Runtime
- 不做多 Agent 协作
- 不做长期记忆或 RAG
- 不自动写入 docs/tasklists/ 目录

## 关键变更

1. 流式协议新增 `agent_step_start` / `agent_step_end` chunk
2. Agent Runtime 实现入口检测 + 状态机
3. 确定性校验工具检查 tasklist 结构完整性
4. 最多一次自动修正循环

## 测试计划

- 验证 `/tasklist` + `@docs://versions/*.md` 入口检测
- 验证草稿 v1 → 校验 → 修正 → 校验 → 最终输出链路
- 验证前端 AgentTracePanel 正确渲染步骤
- 验证无版本方案引用时的明确提示

## 交付结果

- Agent Runtime 可控链路跑通
- 前端可观察 Agent 执行过程
- 最终输出可复制 tasklist 草稿
