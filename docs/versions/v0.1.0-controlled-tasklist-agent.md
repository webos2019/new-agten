# v0.1.0 — 受控 Tasklist Agent

## 版本目标

- 实现第一个受控 Agent：Tasklist Agent
- 基于 `/tasklist` + `@docs://versions/*.md` 显式入口
- 生成任务清单草稿，不自动写入文件
- 确定性质量门校验结构完整性

## 非目标

- 不做通用 Agent
- 不做开放式 Planner
- 不做多 Agent
- 不做自动文件写入
- 不做 RAG / chunking / indexing

## 关键变更

1. Composer 支持 `/tasklist` 斜杠命令
2. 资源菜单支持 `@docs://versions/*.md` 引用
3. Chat Orchestrator 新增 Agent 受控分支
4. AgentState 管理草稿 v1/v2、校验结果、修正次数
5. planExtract 步骤提取版本方案结构化依据

## 测试计划

- 入口检测：缺少版本方案引用时给出明确提示
- 草稿生成：模型基于 planExtract 结果生成 tasklist
- 结构校验：检查标题、来源、步骤、验收项
- 修正循环：最多一次自动修正
- 最终输出：可复制草稿 + 校验结论 + 人工确认点

## 交付结果

- Tasklist Agent 完整链路跑通
- Agent 边界可控、步骤可观察、输出可接力
